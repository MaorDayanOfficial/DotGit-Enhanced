Privacy Policy

DotGit Enhanced — Browser Extension

Last updated: March 5, 2026

Overview

DotGit Enhanced ("the extension") is a browser extension for security researchers that detects exposed version control directories on websites. This privacy policy explains what data the extension handles, how it is stored, and your rights regarding that data.

The short version: All data stays on your device. Nothing is collected, transmitted, or shared with anyone — including the developer. There are no analytics, no tracking, no external servers, and no accounts.

Data the Extension Accesses Temporarily

Web History

The extension observes website origins (e.g., https://example.com) via the webRequest API to determine when you navigate to a new site. Only the origin is recorded — not the full URL path, page title, or browsing duration. This list is stored locally to prevent scanning the same origin twice. It is never transmitted externally.

Website Content (User-Initiated Downloads Only)

When you explicitly click the download button in the extension popup, the extension sends HTTP GET requests to the target website to fetch exposed Git objects (e.g., .git/HEAD, .git/objects/...). These objects are decompressed and processed in memory to reconstruct source files, then packaged into a ZIP file saved to your local filesystem via the browser's download API.

This website content is never transmitted to any external server. It exists only in browser memory during processing and in the resulting local ZIP file.

Data the Extension Does NOT Collect

  • Personally identifiable information — no names, emails, addresses, or IDs
  • Authentication information — no passwords or credentials
  • Financial information — no payment or transaction data
  • Health information — none
  • Location data — no IP addresses, GPS coordinates, or geolocation
  • Personal communications — no emails, messages, or chat data
  • User activity tracking — no clicks, mouse movements, scrolling, or keystrokes
  • Analytics or telemetry — no usage statistics are collected or transmitted

Third-Party Services

The extension does not use any third-party services, APIs, analytics platforms, or external servers. There is no Google Analytics, no crash reporting, no CDN, and no remote code execution. All JavaScript libraries (Pako, JSZip) are bundled locally within the extension package.

Data Sharing and Transfer

The extension does not sell, transfer, share, or transmit any user data to any third party under any circumstances. All data remains entirely on the user's device.

Specifically:

  • No data is sold or transferred to third parties
  • No data is used for advertising, profiling, or marketing purposes
  • No data is used to determine creditworthiness or for lending purposes
  • No data is transmitted to the developer or any external server

Data Security

All data is stored using Chrome's built-in chrome.storage.local API, which is sandboxed to the extension and inaccessible to websites or other extensions. Downloaded ZIP files are saved to the user's standard download directory with the same protections as any browser download.

User Control and Data Deletion

You have full control over all stored data:

  • Delete individual findings — Click the delete button next to any finding in the extension popup
  • Clear all findings — Use the clear all button in the extension popup
  • Reset settings — Use the reset option in the extension settings page
  • Complete removal — Uninstalling the extension removes all stored data automatically

Children's Privacy

The extension is not directed at children under the age of 13. It does not knowingly collect any personal information from children.
