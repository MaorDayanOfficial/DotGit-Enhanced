For authorized security research only. Unauthorized access to systems is illegal. Always ensure proper authorization. This extension stores visited domains locally to avoid duplicate checks — clear via the trash icon. Use at your own risk.

What It Does

Automatically detects exposed .git repositories, .env files, and other sensitive directories on every website you visit. When found, one click downloads the Git objects and reconstructs the actual source files — delivering a ZIP with both raw data and extracted source code.
